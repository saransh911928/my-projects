Backend:

# npm init -y

# npm install express / npm i express

# npm i -g nodemon

# npm i dotenv

# npm i mongoose

## Run: npm run dev

## API Endpoints:

To run the backend server, use the command:

```npm run dev


GET /workouts --> Get all the workouts doc
POST /workouts --> Create a new workout doc
GET /workouts/:id --> Get a single workout doc
DELETE /workouts/:id --> Delete a single workout doc
PATCH /workouts/:id --> Updates a single workout doc
